# 📊 Pesquisa de Opinião - TudoWeb

Este projeto foi desenvolvido em Python com o objetivo de coletar e analisar a opinião de clientes sobre o atendimento de uma empresa.

## 🎯 Objetivo

Realizar uma pesquisa com 50 entrevistados e verificar o nível de satisfação no atendimento.

## 🛠️ Tecnologias

- Python
- GitHub

## 📌 Funcionamento

O programa solicita:
- Nome
- Idade
- Opinião sobre o atendimento:
  - 1: EXCELENTE
  - 2: BOM
  - 3: RUIM

Utiliza estrutura de repetição **FOR** e estrutura de decisão **IF/ELIF**.

## 📊 Resultado

Ao final, o sistema exibe:
- Quantidade de respostas EXCELENTE
- Quantidade de respostas RUIM
